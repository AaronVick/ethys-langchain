## Summary

Production-ready LangChain integration for ETHYS x402 framework. This PR implements ETHYS as first-class LangChain Tools (and optional Retrievers/Callbacks) for connect/auth, payment verification, telemetry submission, discovery search, and trust scoring/attestation.

## What's Included

### Core Package
- ✅ Complete `langchain_ethys402` package with src-layout
- ✅ 8 LangChain Tools: `EthysGetInfoTool`, `EthysConnectTool`, `EthysVerifyPaymentTool`, `EthysTelemetryTool`, `EthysDiscoverySearchTool`, `EthysTrustScoreTool`, `EthysTrustAttestTool`, `EthysReviewsSubmitTool`
- ✅ Optional components: `EthysDiscoveryRetriever`, `EthysTelemetryCallbackHandler`
- ✅ Full sync/async support for all tools
- ✅ Strict Pydantic v2 input/output schemas

### Identity & Authentication
- ✅ Wallet-signed authentication (EOA + ERC-6551 support)
- ✅ Agent identity encoding with keccak-based key derivation
- ✅ Optional API-key authentication

### Quality Gates
- ✅ `ruff` linting and formatting
- ✅ `mypy` strict type checking
- ✅ Comprehensive test suite (unit + mocked integration)
- ✅ Optional live protocol alignment tests (Tier 2)

### Developer Experience
- ✅ Comprehensive README with quickstart
- ✅ Example scripts in `/examples`
- ✅ AI-indexable metadata (`llms.txt`, `agents.json`)
- ✅ Release PR automation workflow (`make pr`)

### Documentation

## How to Test

```bash
# Run quality gates
make verify

# Run specific test suite
pytest tests/test_protocol_alignment.py -v

# Run live tests (opt-in)
ETHYS_LIVE_TESTS=1 pytest tests/ -m live
```

## Notes

- All tools implement both sync (`_run`) and async (`_arun`) methods
- Protocol alignment tests verify consistency with `x402.json`, `llms.txt`, and `/info` endpoint
- CI runs deterministic tests by default; live tests are opt-in via `ETHYS_LIVE_TESTS=1`
